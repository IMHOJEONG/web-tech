# 2026-06-12 docs 사이트 메타데이터 한국어 카피 정리

## 작업 내용

- `apps/docs`의 한국어 사이트 메타데이터 카피를 더 자연스럽게 다듬었다.
- 기존에는 영어 슬로건을 그대로 둔 상태라 한국어 메타 설명과 결이 조금 어긋나 있었다.
- 이번에는 브랜드명은 유지하고, `title`, `description`, `ogTitle`, `ogDescription`를 한국어 기술 아카이브 톤으로 정리했다.

## 변경 기준

- 직역투보다 개발자 블로그/기술 아카이브에 가까운 표현을 우선
- “docs 경험” 같은 번역체보다 “기술 아카이브”, “기술 메모”, “지식 베이스”처럼 실제 서비스 성격이 보이는 표현 사용
- 브랜드 문구는 과장보다 구조적이고 차분한 톤 유지
