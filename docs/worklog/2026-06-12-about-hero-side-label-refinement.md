# 2026-06-12 About hero 보조 라벨 문구 정리

## 작업 내용

- `About` hero 우측 보조 라벨 문구를 더 자연스럽게 다듬었다.
- 기존 `// 운영 원칙`, `// 2024부터 기록 중`은 화면 톤과 크게 어긋나진 않았지만, 실제 소개 문맥에서는 조금 딱딱하고 임시 문구처럼 느껴질 수 있었다.

## 변경 내용

- `// 운영 원칙` → `// 만들고 기록하는 기준`
- `// 2024부터 기록 중` → `// 2024년부터 이어온 아카이브`

## 판단 이유

- `About` 섹션은 서비스 상태보다 사람과 아카이브의 성격을 보여주는 영역에 가깝다.
- 그래서 “운영”보다 “만들고 기록하는 방식”, “언제부터 이어왔는지”가 더 자연스럽게 읽히도록 맞췄다.
