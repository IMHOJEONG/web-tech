# 2026-05-26 한국어 hero 카피 다듬기

## 요약

- `/` root landing과 `/about` hero를 중심으로, 직역투가 강하던 한국어 카피를 더 자연스럽게 다듬었다.
- 이번에는 headline, eyebrow, CTA, section title처럼 첫 인상에 직접 닿는 문구를 우선 정리했다.

## 반영 파일

- `apps/docs/shared/message/ko.json`
- `docs/todo/todo.md`

## 이번에 바꾼 방향

- 영어 카피를 직역한 느낌이 강한 문장은 더 짧고 자연스러운 한국어 문장으로 조정
- `SYSTEM ENGINEERING & DESIGN`, `THEMATIC FOUNDATIONS`, `LATEST NOTES` 같은 루트 landing 레이블도 한국어 맥락에 맞게 정리
- `/about` hero는 “기술 스택” 같은 어색한 표현보다, 이 프로젝트가 다루는 실제 범위인 웹, 제품, 인터페이스를 드러내는 쪽으로 조정
- About pillar도 headline과 설명이 조금 더 한국어다운 흐름으로 읽히게 수정

## 메모

- 이번 작업은 `landing / hero 계열 카피` 항목 전체 완료가 아니라, 우선순위가 높았던 `/` root landing과 `/about` hero를 먼저 정리한 단계다.
- `/web`, `/mobile`, `/ui-ux` 허브와 기타 섹션 카피는 후속으로 더 다듬을 수 있다.
