# 2026-06-12 About 프로필 플레이스홀더 제거

## 작업 내용

- `About` 페이지 프로필 카드에 있던 `AV` 플레이스홀더 블록을 제거했다.
- 해당 블록은 실제 아바타나 의미 있는 아이콘이 아니라 장식용 placeholder에 가까워서, 현재 레이아웃에서는 정보보다 장식이 먼저 보이는 상태였다.
- 프로필 카드 상단은 이름과 역할만 남기고 더 단정하게 정리했다.

## 판단 이유

- 실제 프로필 이미지가 없는 상태에서 이니셜 박스를 두면 완성된 프로필 UI처럼 보이기보다 임시 상태처럼 보일 수 있다.
- 현재 `About` 카드의 핵심은 사람 정보와 링크이므로, 시선 분산 요소를 줄이는 편이 자연스럽다.
